# Background * Wanted to go SOA * Stability, scala & faster iteration * Obstacles * Minimal experience with Scala * Can't risk creating more problems * Solution * Use Play as stepping stone * **This talk** * **Build your first microservice**
# What we will cover * Build a URL shortener * A few lines of code for each: * JSON parsing and serialization * Persist data in MySQL * Anorm * Caching * Composing services

Setup Play project

Create a new project with Typesafe Activator CLI

activator new url-shortener play-scala

Add two extra dependencies to build.sbt

"com.typesafe.play" % "anorm" %% "2.3.9",
"mysql" % "mysql-connector-java" % "5.1.35"
